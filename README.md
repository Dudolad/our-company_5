our-company

Проєкт створений для моделювання структури бібліотеки.

Основні класи:
Publication – базовий клас видання.
Book – книга (назва, автор, рік, резюме).
Magazine – журнал (назва, автор, рік, статті).
Library – бібліотека, що зберігає видання.
PublicationException, BookException, EditionYearException – власні класи помилок.

Функціональність:
Додавання видань у бібліотеку.
Пошук видання за назвою.
Виведення списку всіх видань.
Перевірка порожніх назв та авторів.
Перевірка року видання через виключення.
Перевірка в main:
Створення бібліотеки.
Додавання книги та журналу.
Виклик виключень і їх обробка.

Проєкт створений для тренування ООП, роботи з виключеннями та командної роботи через Git.
